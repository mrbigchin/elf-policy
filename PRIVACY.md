# Elf Bot Privacy Policy

Effective date: June 13, 2026

Elf is a private Discord bot used for moderation, event management, submission tracking, and community utilities in the Anime Expeditions Discord server.

## Data We Collect

Elf may collect and store limited Discord data needed to operate server features:

- Discord user IDs
- Discord message IDs
- Discord channel IDs
- Discord role IDs used for permissions
- Submitted event links
- Attachment URLs for submitted artwork
- Invite codes and invite use counts
- Event progress data, such as boss HP, contribution counts, approved submissions, and pending submissions
- Moderation or automation state needed to prevent duplicate submissions or repeated bot actions

Elf does not intentionally store general chat message content. In specific event or moderation channels, Elf may read message content to validate links, detect duplicate submissions, suppress embeds, or remove disallowed command messages. Submitted URLs may be stored for duplicate detection.

## How We Use Data

Collected data is used only to operate server features, including:

- Tracking event participation and boss event progress
- Reviewing and approving user submissions
- Detecting duplicate submission links
- Tracking invite-based event progress
- Removing invalid messages in configured channels
- Preventing bot commands from being used outside allowed command channels
- Maintaining starboard and suggestion highlight features

We do not sell user data. We do not use user data for advertising. We do not use message content or other Discord data to train machine learning or AI models.

## Data Storage

Elf stores operational data in files or storage connected to the bot's hosting service. This data is kept only as long as needed for server moderation, event tracking, and bot functionality.

Access to stored data is limited to server staff or bot maintainers who need it to operate the bot.

## Data Sharing

Elf does not share collected data with third parties except where required by the bot's hosting provider or Discord itself to provide the service.

## Data Removal

Users may request removal of stored data related to their submissions or event participation by contacting Anime Expeditions server staff. Some data may need to be retained temporarily for moderation, abuse prevention, or event integrity.

## Contact

For questions about this policy or data handled by Elf, contact the Anime Expeditions server staff through the Discord server.
